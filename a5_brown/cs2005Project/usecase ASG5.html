Title: Make Quiz

Primary Actor: Instructor

Stakeholders and Interests:

Instructor: wants to create questions for students

Student: wants to receive questions in a usable format

Technician: wants changes to be traceable 

Precondition: User with Instructor privileges logged in already.

Trigger: Instructor requests to create a new quiz. 

Main Scenario:

  1. Instructor selects option to create a new quiz

  2. SuD displays page to create quiz

  3. Instructor specifies name of quiz, people in the quiz, weight on final grade, etc.

  4. SuD records quiz and proceeds to reading questions

  5. Instructor adds a question to the quiz, specifying answers, weight

  6. SuD records question info, and gives option to submit quiz

  7. Instructor repeats 4-6 until quiz is finished, then submits quiz

  8. SuD records finalized quiz

Extensions:

7a. Edit quiz 

7a1. Instructor accesses quiz edit function

7a2. SuD returns to 4 in Main Scenario

7b. Push quiz to students

7b1. Instructor accesses quiz push function

7b2. SuD invokes Notify Student use case
